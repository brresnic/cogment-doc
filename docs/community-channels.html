Community

The best way to interact with the Cogment community is to use Discord and join our server. If you're unfamiliar with Discord, it's a community tool allowing people to regroup, chat via text and voice and interact, that can be used either through any web browser, or through its dedicated client applications for computers and mobile devices.

Cogment Discord Server​

To join our Discord server, simply click on the link below. If you have the dedicated client application, it will open it there, if not, it will use your web browser.
